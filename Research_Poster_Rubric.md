# Research Poster Rubric

**Total Score: /50**

---

## Part I: Appearance & Organization

**Maximum Points Available: 15 | Subtotal: ___/15**

Scale: 0 = Strongly Disagree · 1 = Disagree · 2 = Agree · 3 = Strongly Agree

| Criterion | 0 | 1 | 2 | 3 |
|-----------|---|---|---|---|
| **Readability** — The poster text is clearly readable from a distance of 3 to 4 feet. | | | | |
| **Figures, Tables & Diagrams** — The figures, tables and diagrams are clear and legible and add to the effectiveness of the poster's message. | | | | |
| **Design & Color Scheme** — Design is cohesive and color scheme is not distracting. | | | | |
| **Format & Grammar** — Format is consistent with no spelling or grammatical errors. | | | | |
| **Organization & Flow** — Poster appears well-organized and information flows naturally from the introduction through the conclusion. | | | | |

---

## Part II: Content

**Maximum Points Available: 35 | Subtotal: ___/35**

Scale: 0 = Strongly Disagree · 1 = Disagree · 2 = Somewhat Disagree · 3 = Neither Agree nor Disagree · 4 = Agree · 5 = Strongly Agree

| Criterion | 0 | 1 | 2 | 3 | 4 | 5 |
|-----------|---|---|---|---|---|---|
| **Intro/Significance/Need/Research Question** — Sufficient background information is provided to establish the significance of the problem being addressed. | | | | | | |
| **Methods** — Methods are sufficiently detailed to allow an understanding of the materials, instruments and procedures used. | | | | | | |
| **Results** — Results address the research question and are presented in a clear and meaningful fashion (appropriate use of figures, tables, and descriptive text). | | | | | | |
| **Discussion** — The significance of the findings is discussed. | | | | | | |
| **Conclusions/Recommendations** — Appropriate conclusions are drawn and recommendations for follow-on or future work provided. | | | | | | |
| **References** — References are provided and properly cited. | | | | | | |
| **Acknowledgements** — Acknowledgements of support from outside agencies and/or others who contributed to the project are provided. | | | | | | |
